Component Selection
===================

Module containing component selection methods and the ComponentSelect class.

The component selection step occurs after the computation of the invariant components.
Several methods are already implemented. If you want to call ICS with another method,
you would need to create a ComponentSelect object, as detailed in the :ref:`custom_compselect` tutorial.
